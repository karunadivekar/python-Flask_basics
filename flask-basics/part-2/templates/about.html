About This Flask Application

This page is also being served from an HTML file in the templates/ folder, just like the Home page.

What happens when you open /about?

When you visit this URL, Flask:

  1. Receives your browser's request
  2. Matches the URL to @app.route('/about')
  3. Calls the about() function
  4. Loads templates/about.html
  5. Sends this HTML back to your browser
