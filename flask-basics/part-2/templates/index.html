Welcome to Flask Templates!

This page is being served from an HTML file in the templates/ folder.

What's happening?

When you visit this URL, Flask:

  1. Receives your browser's request
  2. Matches the URL to the @app.route('/')
  3. Calls the home() function
  4. Loads templates/index.html
  5. Sends this HTML back to your browser

Why Use Templates?

  • Separation of Concerns: Python handles logic, HTML handles presentation
  • Easier to Maintain: Edit HTML without touching Python
  • Reusability: Same template for different data
  • Jinja2 Power: Python-like logic in HTML
